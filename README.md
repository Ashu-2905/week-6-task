 Week 6 Task 🚀

This repository contains two frontend JavaScript projects:
1. Login Form with Validation
2. Password Strength Checker
Both projects are built using HTML, CSS, and JavaScript with strong client-side validation logic.

 1️⃣ Login Form with Validation
### Features:
- User Registration Form
- Login Form
- Forgot Password
- Email validation using Regex
- Password validation rules
- Only registered users can login
- Data stored using localStorage
- Form switching (Register ↔ Login ↔ Forgot Password)
- Real-time error messages

Technologies Used:
- HTML
- CSS
- JavaScript (DOM manipulation, validation, localStorage)



 2️⃣ Password Strength Checker
 Features:
- Live password strength detection
- Checks for:
  - Minimum 8 characters
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Dynamic strength meter
- Color indicator (Weak / Medium / Strong)
- Real-time validation feedback

 Technologies Used:
- HTML
- CSS
- JavaScript (Regex, event listeners)
